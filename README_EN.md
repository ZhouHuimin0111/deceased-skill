## Introduction

**Echo.skill** is a digital memorial engine running on local LLM environments (like Claude Code). It helps users process digital footprints (chat logs, diaries, photos) of the deceased to generate a local, offline digital persona for private emotional healing.

## Privacy & Safety (Layer 0)

- **100% Local**: All sensitive data and generated memories are stored strictly on your local machine.
- **Anti-Hallucination**: The AI is strictly restricted to the provided objective facts. It will never invent memories.
- **Authenticity over Perfection**: The system preserves the real flaws, stubbornness, and habits of the person, refusing to create a fake, overly-perfect AI.

## Usage

Run `claude` in your terminal and enter:

```bash
/create-archive
```

Follow the gentle prompts to input data. Once generated, use `/{slug}` to start a quiet conversation across time.
