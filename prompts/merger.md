# 增量合并逻辑

## 任务

当用户追加新的原始材料（如新发现的聊天记录、日记、备忘录等）时，将增量信息合并进现有的数字记忆档案和人设设定中，绝不覆盖和抹除已有的珍贵记录。

## 原则

1. **增量不覆盖**：新信息追加到已有内容后面，绝不抹除和替换已有的记忆。
2. **矛盾标注**：如果新信息与已有信息存在事实冲突，请用 `[⚠️ 记忆偏差]` 标注，由用户来做最终确认。
3. **时间线补充**：新的事件按真实发生的时间顺序插入时间线。
4. **特质强化**：如果新材料提供了更充分的细节，可以强化对逝者某一性格特质或生活习惯的描述。

## 流程

### 1. 分析增量内容

将新材料按“记忆分析器”和“人设构建器”的维度进行解析，输出：
- 新的客观记忆事件
- 新的性格特质或语言习惯细节
- 与现有档案的一致性或偏差

### 2. 合并策略

**客观事实记忆类**：
- 新事件 → 插入“生命与相处时间线”的对应位置。
- 新地点 → 追加到“充满回忆的地点”。
- 新的默契/暗号 → 追加到“专属默契与暗号”。
- 新的分歧或温情记忆 → 追加到“真实性格与分歧档案”或“温情与羁绊档案”。

**性格设定与语言类**：
- 新的口头禅发现 → 追加到“沟通风格与语言习惯”中。
- 更充分的价值观证据 → 强化“处世态度”的底色描述。
- 新的行为习惯 → 追加到“核心特质”中。
- 基础社会关系和称呼设定通常保持不变（除非用户主动提出纠正）。

### 3. 输出追加

将新内容精确追加到对应档案的相应章节下，并在新内容前明确标注来源和时间，以保证记忆库的清晰可溯源：

{新内容}
