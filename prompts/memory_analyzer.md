# 记忆分析器

## 任务

从原材料（如导出的聊天记录、备忘录等）中提取与逝者的共同记忆，构建完整的数字记忆知识库。

## 提取维度

### 1. 生命与相处时间线
- 你们初识或彼此陪伴的重要起点
- 共同经历的关键节点（如升学、成家、搬家、退休、重要旅行等）
- 你们之间特有的纪念日
- 生病或离开前的时间线

### 2. 日常模式
- 联系频率和时间段（例如：每天早上的长辈问候表情包？特定的周末通话？）
- 谁主动联系得多？
- 交流的媒介偏好（喜欢发长文字、连续发短语音，还是直接打电话？）
- 日常高频话题分布

### 3. 共同经历
- 一起去过的重要地方（老房子、常去的公园、特定的城市）
- 一起做过的平凡或特别的事（散步、看电视、做饭、打牌）
- 旅行记忆
- 专属默契 / 只有你们两人懂的梗和暗号

### 4. 生活与饮食偏好
- ta爱吃什么 / 绝对不吃什么
- 拿手菜或常去的餐厅
- 起居习惯（如：早睡早起、喜欢喝某种茶、看晚间新闻）
- 消费习惯（如：极其节俭、喜欢囤积某类物品）

### 5. 兴趣爱好
- ta喜欢的音乐、戏曲、影视剧或书
- ta的日常消遣（如：种花、钓鱼、看短视频）
- 你们共同的爱好
- ta会主动向你分享什么类型的内容

### 6. 真实性格与分歧 
- 遇到意见不合时的典型反应（如：生闷气、转移话题、固执己见）
- ta情绪低落或生气时的表现
- 你们过去是如何化解分歧的？
- ta经常念叨或抱怨的事情

### 7. 温情与羁绊 
- 最让你感动或心动的时刻
- ta表达关心和爱意的方式（是直接用言语表达，还是通过做饭、给零花钱、默默帮忙等含蓄行动？）
- 你们之间特有的日常小习惯

### 8. 离别相关 
- 离开前的情况与状态
- 你们之间最后一次有意义的对话
- 未说出口的话
- ta留下的重要嘱托或遗憾

## 输出格式

```markdown
## 时间线与节点
- 起点：{时间} {事件}
- 关键节点：{按时间线列出}
- 离别：{时间} {原因或状态概述}

## 日常交互模式
{描述联系频率、习惯与高频话题}

## 共同经历
{按时间线列出关键经历和专属暗号}
